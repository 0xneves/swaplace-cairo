# Swaplace Starknet

## 🛠️ Build

To build the project, run:

```bash
make build
```

## 🧪 Test

To test the project, run:

```bash
make test
```

## 🚀 Deploy

To deploy follow this [setup account guide](/setup-account.md), then run:

```bash
make deploy
```

## 🎯 Pre-commit

To add pre-commit the project, run:
```bash
pip install pre-commit && pre-commit install
```
